# GitStudy

```
git init : 초기화하기
git config --global user.name "이름" : 유저 이름 설정하기
git config --global user.email "이메일" : 유저 이메일 설정하기
git status : 현재 상태 확인하기
git add : 변경사항 더하기
	add 뒤에 -A, --all, ., *이 붙으면 모든 변경사항을 반영한다는 의미이다.
git commit : 변경사항 반영하기
git log : 반영사항 확인하기
git reset : 이전 상태로 되돌리기
git revert : 이전 상태로 되돌리기
```
